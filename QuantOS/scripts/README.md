# Operational Scripts
Helper scripts for deployment and emergencies.

* **Installer**: `install_mac.command` for automated environment setup.
* **Updater**: `update.py` to synchronize system files with the master repository.
* **Security Audit**: `security_sweep.py` for hardcoded secret detection.
* **Testing**: `debug_backtest.py` for isolated simulation troubleshooting.
* **Emergency Sell**: Scripts to liquidate positions immediately.
